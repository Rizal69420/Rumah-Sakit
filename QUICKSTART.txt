QUICK START GUIDE - Rumah Sakit Sehat Sentosa
=============================================

📋 STEP-BY-STEP SETUP

1. COPY PROJECT
   - Copy folder "Rumah Sakit" ke C:\xampp\htdocs\

2. BUAT DATABASE
   - Buka browser: http://localhost/Rumah%20Sakit/database/create_db.php
   - Tunggu sampai selesai (akan ada pesan "Database berhasil dibuat!")
   - Catat login credentials yang ditampilkan

3. AKSES WEBSITE
   - Homepage: http://localhost/Rumah%20Sakit/
   - Login/Register: http://localhost/Rumah%20Sakit/auth.php

🐛 TROUBLESHOOTING

PROBLEM: "Connection Failed" saat akses auth.php
SOLUTION: Jalankan http://localhost/Rumah%20Sakit/database/create_db.php

PROBLEM: Blank page / Error 500
SOLUTION: Cek PHP error logs di C:\xampp\logs\php_error.log

PROBLEM: Database sudah ada, ingin reset?
SOLUTION: 
  1. Drop database: phpMyAdmin > Databases > Select > Drop
  2. Jalankan create_db.php lagi

PROBLEM: Appointment tidak tersimpan
SOLUTION: Pastikan login sebagai user, bukan admin
