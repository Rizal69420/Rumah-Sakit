👥 USER WORKFLOW:

1. Register/Login di auth.php
2. Explore halaman (Home → About → Departments → Services)
3. Lihat dokter di halaman Doctors
4. Click "Booking" untuk membuat appointment
5. Pilih dokter, layanan (opsional), tanggal & waktu
6. Submit form
7. Lihat appointment Anda di tab "Appointments"
8. Status appointment akan update real-time

---

👨‍💼 ADMIN WORKFLOW:

1. Login dengan admin@sehat-sentosa.com / admin123
2. Redirect ke admin dashboard otomatis
3. Dashboard menampilkan:
   - 6 stat cards (appointment, pending, dokter, user, pesan, layanan)
   - Real-time appointment status overview
   - Tabel appointment terbaru
   - Auto-refresh setiap 30 detik

4. Gunakan sidebar untuk:
   a. Kelola Dokter
      - Lihat daftar dokter
      - Tambah dokter baru
      - Edit dokter (spesialisasi, pengalaman, status)
      - Hapus dokter
      
   b. Kelola Departemen
      - Lihat daftar departemen
      - Tambah departemen baru
      - Edit departemen
      - Hapus departemen
      
   c. Kelola Layanan
      - Lihat daftar layanan & harga
      - Tambah layanan baru
      - Edit layanan & harga
      - Hapus layanan
      
   d. Kelola Appointment
      - Lihat semua appointment dari pasien
      - Update status appointment (pending → confirmed → completed)
      - Edit appointment details
      - Hapus appointment
      
   e. Kelola User
      - Lihat daftar user yang register
      - Lihat jumlah appointment per user
      - Delete user (kecuali admin)
      
   f. Kelola Pesan
      - Lihat pesan dari pengunjung
      - Mark sebagai read
      - Balas pesan dari admin
      - Delete pesan
